App is built with Korlin and MVI architecture
Those libraries are used extinsively in the app
  - Apollo - Graphql
  - Dependancy injection by Hilt
  - AndroidX libs
  - Room DB
  - Flashbar
  - Kotlin coroutines
  - Unit testing
